# 🌟 Anuradha | Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Dynamic Hero** — Animated text intro with smooth transitions
- **About Me** — Brief personal introduction
- **Skills Section** — Showcasing 5 key technical skills with interactive cards
- **Certifications** — IBM, Hackathon, DBMS, and DSA certifications
- **Achievements** — Hackathon participation & IEEE membership
- **Languages** — Multilingual proficiency display
- **Contact Form** — Get-in-touch section with email & location info

## 🎨 Design

- Dark-themed with blue/purple gradient aesthetic
- Glassmorphism card effects
- Custom cursor glow & animated background blobs
- Smooth hover micro-animations
- Fully responsive layout

## 🛠️ Tech Stack

- HTML5
- CSS3 (Vanilla — no frameworks)
- JavaScript (Vanilla)
- [Lucide Icons](https://lucide.dev/)
- [Google Fonts — Outfit & Inter](https://fonts.google.com/)

## 📁 Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # All styles & animations
├── script.js           # Interactive JavaScript
└── anuradha_profile.png # Profile image
```
